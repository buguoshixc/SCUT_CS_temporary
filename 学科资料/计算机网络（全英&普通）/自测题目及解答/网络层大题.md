### 【大题一】综合寻址、子网划分与 ICMP（模型一、二、三）

【题目背景】

某公司获得一个地址块 192.168.1.0/24。网络拓扑为：

主机 A（部门 1） $\rightarrow$ 路由器 R1 $\rightarrow$ 路由器 R2 $\rightarrow$ 主机 B（外网）。

1. **部门 1** 有 120 台主机。
    
2. **R1** 到 **R2** 之间的点对点链路使用该地址块中剩下的空间。
    

**【问题】**

1. **子网划分**：请给出部门 1 的网络地址、子网掩码及可用的 IP 地址范围。
    
2. **转发流程**：当主机 A 发送一个 IP 数据报给主机 B 时：
    
    - 在 R1 到 R2 的链路上，该数据报的**源 IP** 和 **目的 IP** 是否变化？
        
    - 在 R1 到 R2 的链路上，该数据报的**源 MAC** 和 **目的 MAC** 分别是什么（用描述性文字回答）？
        
3. **差错处理**：若主机 A 发出的数据报 TTL=1，说明接下来会发生什么？
    

---

#### ✅ 【大题一：参考答案】

1. **子网划分**：
    
    - **网络地址**：`192.168.1.0`（因为 120 台主机需要 7 位主机位，$2^7-2=126 > 120$）。
        
    - **子网掩码**：`255.255.255.128` (即 `/25`)。
        
    - **可用 IP 范围**：`192.168.1.1` ~ `192.168.1.126`（排除 .0 网络地址和 .127 广播地址）。
        
2. **转发流程**：
    
    - **IP 地址**：**不变化**。源 IP 依然是 A，目的 IP 依然是 B。
        
    - **MAC 地址**：**源 MAC** 为路由器 R1 靠近 R2 端的接口 MAC 地址；**目的 MAC** 为路由器 R2 靠近 R1 端的接口 MAC 地址。
        
3. **差错处理**：
    
    - 数据报到达 R1 后，R1 执行 TTL 减 1 操作，此时 **TTL=0**。
        
    - R1 将**丢弃**该数据报，并向源主机 A 发送 **ICMP 时间超过报文**（Type 11, Code 0）。

### 【大题二】NAT 转换与路由查表（模型三、四）

【题目背景】

内网主机 A (192.168.1.10) 通过 NAPT 网关（公网 IP 为 202.1.1.1）访问外网服务器 B (8.8.8.8)。

此时网关的 NAT 转换表如下：

| 内网 IP : 端口 | 外网 IP : 端口 |

| :--- | :--- |

| 192.168.1.10 : 5000 | 202.1.1.1 : 8000 |

同时，网关路由器收到一个去往 `8.8.8.8` 的回包，其路由表条目如下：

1. `0.0.0.0/0` 下一跳 `202.1.1.254`
    
2. `8.0.0.0/8` 下一跳 `202.1.1.5`
    

**【问题】**

1. **NAT 填表**：当服务器 B 回复 A 时，该回包到达网关（公网侧）时，其首部的**目的 IP** 和**目的端口**各是什么？
    
2. **NAT 转发**：网关收到回包后，根据转换表，会将包的首部修改为什么？
    
3. **最长前缀匹配**：当网关需要将请求包发往服务器 `8.8.8.8` 时，根据路由表，它会选择哪一个下一跳地址？为什么？
    

---

#### ✅ 【大题二：参考答案】

1. **回包到达网关时**：
    
    - **目的 IP**：`202.1.1.1`（NAT 网关的公网 IP）。
        
    - **目的端口**：`8000`（转换表分配的外网端口）。
        
2. **网关还原后**：
    
    - **目的 IP**：`192.168.1.10`。
        
    - **目的端口**：`5000`。
        
3. **路由选择**：
    
    - 选择**下一跳 `202.1.1.5`**。
        
    - **理由**：根据**最长前缀匹配原则**，`8.8.8.8` 与 `8.0.0.0/8` 匹配（掩码长度 8），而默认路由 `0.0.0.0/0` 掩码长度为 0。8 位比 0 位更精确，故选之。
        

---

### 📝 考前最后三句话：

1. **算子网**：网络地址一定是块大小（256-掩码）的倍数。
    
2. **看寻址**：IP 全程不变，MAC 每一跳都变，NAT 只有过网关才变。
    
3. **论 ICMP**：报错一定发给源 IP，TTL=0 报超时，找不到路报不可达。
