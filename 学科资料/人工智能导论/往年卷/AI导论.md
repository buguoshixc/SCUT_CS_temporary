### **二、简答题答案要点**  
1. **回溯 vs 图搜索**  
   - **回溯**：深度优先，不记录路径，可能重复访问节点。  
   - **图搜索**：记录已访问节点（Closed表），避免重复，保证完备性。  

2. **不确定性表现**  
   - 证据不确定性（如传感器误差）  
   - 规则不确定性（如专家经验概率）  
   - 推理结论不确定性（多路径传播）。  

3. **泛化能力**  
   模型在**未见数据**上准确预测的能力，反映学习本质（归纳规律，非记忆数据）。  

4. **分类 vs 聚类 vs 回归**  
   | **类型** | **输出**       | **监督** | **目标**                |  
   |----------|----------------|----------|-------------------------|  
   | 分类     | 离散类别       | 是       | 学习决策边界            |  
   | 聚类     | 无标签分组     | 否       | 发现数据内在结构        |  
   | 回归     | 连续数值       | 是       | 拟合输入-输出映射关系   |  

5. **知识表示方法**  
   - **谓词逻辑**：形式化规则（如P(x)→Q(x)）  
   - **产生式**：IF-THEN规则  
   - **语义网络**：节点与有向弧表示实体关系  
   - **框架**：结构化槽填充（如“汽车”框架含颜色、品牌等槽）  

6. **聚类 vs 分类 & K-means步骤**  
   - **差异**：聚类无标签，分类有标签。  
   - **K-means步骤**：  
     1. 随机选K个质心  
     2. 分配样本到最近质心  
     3. 更新质心为簇均值  
     4. 重复2-3直至收敛。  

7. **不确定性推理**  
   - **定义**：在证据、规则不完全可靠下的推理。  
   - **基本问题**：不确定性表示、传播、合成、结论决策。  

8. **逻辑表示法优缺点**  
   - **优点**：严密、可演绎推理。  
   - **缺点**：表达效率低、难以处理模糊知识。  

9. **AI vs 机器学习 vs 神经网络**  
   - **AI**：宏观目标（机器智能）  
   - **机器学习**：AI子领域（数据驱动学习）  
   - **神经网络**：机器学习模型（仿生结构）。  

10. **贝叶斯定理**  
    - **公式**：P(A|B) = P(B|A)P(A)/P(B)  
    - **解决**：逆概率问题（从P(症状|疾病)推P(疾病|症状)）  
    - **刻画**：因果与相关并存（需因果图辅助区分）。  

11. **AI搜索 vs 数据库检索**  
    - **AI搜索**：状态空间探索（路径优化，如A*算法）  
    - **数据库检索**：索引匹配（精确查询，无路径代价）。  

12. **梯度下降步骤**  
    1. 初始化参数θ  
    2. 计算损失函数梯度∇J(θ)  
    3. 更新θ ← θ - α∇J(θ)（α为学习率）  
    4. 重复2-3直至收敛。  

13. **机器学习系统构成与风险**  
    - **构成**：数据→特征提取→模型训练→评估→部署  
    - **不可预测性**：数据偏见导致歧视、黑盒模型决策不可解释。  

14. **符号主义 vs 连接主义**  
    - **符号主义**：基于规则推理（如专家系统）  
    - **连接主义**：基于数据学习（如神经网络）。  

15. **物理符号系统 vs 神经网络**  
    - **符号系统**：离散符号操作（可解释性强）  
    - **神经网络**：连续向量变换（并行计算强，黑盒性）。  

16. **DBSCAN思想与缺点**  
    - **思想**：基于密度，核心点（邻域内点数≥MinPts）扩展簇。  
    - **缺点**：对密度不均数据敏感，高维数据距离失效。  

17. **反演法求答案**  
    1. 将问题¬A加入知识库  
    2. 用归结原理推导矛盾  
    3. 若矛盾，则A为真。  

18. **交叉验证法**  
    - **步骤**：数据分K份，轮流用K-1份训练、1份验证。  
    - **用途**：评估模型泛化能力，避免过拟合。  

19. **产生式推理方式**  
    - **正向推理**：从证据触发规则（数据驱动）  
    - **反向推理**：从目标回溯证据（目标驱动）。  

20. **机器学习系统构成与本质**  
    - **构成图**：  
      ```  
      输入数据 → 特征工程 → 模型训练 → 输出预测  
                ↑               ↓  
              评估反馈 ← 验证数据  
      ```  
    - **本质**：归纳规律（非记忆数据），因模型压缩数据为参数化函数。  

21. **BP算法**  
    - **适用结构**：全连接前馈神经网络。  
    - **过程**：  
      1. 前向传播计算输出  
      2. 反向传播误差（输出层→隐藏层）  
      3. 梯度下降更新权重。

### **三、分析计算题答案**  
#### 1. α-β剪枝  
![](https://i.imgur.com/8W7X9Kp.png)  
- **剪枝位置**：节点B的右侧子节点（值≤3 < α=5）。  

#### 2. 贝叶斯网络计算 P(¬C | ¬E)  
**步骤**：  
- 联合概率分解：P(S,C,E) = P(S)P(C)P(E|S,C)  
- 计算 P(¬E)：  
  ```  
  P(¬E) = Σ_{S,C} P(¬E|S,C)P(S)P(C)  
        = P(¬E|S,C)P(S)P(C) + ...  
        = (0.1)(0.4)(0.7) + (0.7)(0.4)(0.3) + (0.5)(0.6)(0.3) + (0.9)(0.6)(0.7)  
        = 0.028 + 0.084 + 0.09 + 0.378 = 0.58  
  ```  
- 计算 P(¬C, ¬E)：  
  ```  
  P(¬C,¬E) = P(¬E|S,¬C)P(S)P(¬C) + P(¬E|¬S,¬C)P(¬S)P(¬C)  
           = (0.7)(0.4)(0.7) + (0.9)(0.6)(0.7)  
           = 0.196 + 0.378 = 0.574  
  ```  
- **结果**：  
  ```  
  P(¬C | ¬E) = P(¬C,¬E) / P(¬E) = 0.574 / 0.58 ≈ 0.9897  
  ```  

#### 3. A*算法 OPEN表变化  
- **初始化**：OPEN=[S(5)]  
- **扩展S**：OPEN=[A(3+4=7), B(2+3=5)] → 扩展B（f最小）  
- **扩展B**：OPEN=[A(7), C(1+2=3)] → 扩展C  
- **扩展C**：OPEN=[A(7), G(0+0=0)] → 路径S→B→C→G（f=3）。  

#### 4. 梯度计算（计算图）  
- **操作**：y = [x₁+x₂, x₂, x₃, x₄]  
- **梯度反传**：  
  ```  
  ∂L/∂x₁ = ∂L/∂y₁ · ∂y₁/∂x₁ = 4 × 1 = 4  
  ∂L/∂x₂ = ∂L/∂y₁ · ∂y₁/∂x₂ + ∂L/∂y₂ · ∂y₂/∂x₂ = 4×1 + (-1)×1 = 3  
  ∂L/∂x₃ = ∂L/∂y₃ = 5  
  ∂L/∂x₄ = ∂L/∂y₄ = 9  
  ```  
- **结果**：∇x = [4, 3, 5, 9]  

#### 5. AND函数神经元参数  
- **要求**：输入[0,0]→0；[1,1]→1；其余→0  
- **解方程**：  
  ```  
  0·w₁ + 0·w₂ + b ≤ 0  → b ≤ 0  
  1·w₁ + 1·w₂ + b > 0  → w₁+w₂+b > 0  
  1·w₁ + 0·w₂ + b ≤ 0  → w₁+b ≤ 0  
  0·w₁ + 1·w₂ + b ≤ 0  → w₂+b ≤ 0  
  ```  
- **可行解**：w₁=1, w₂=1, b=-1.5

#### 7. 决策树预测（PlayTennis=no）

  1. 根节点选**Outlook**（最高信息增益）

#### 8. 归结原理证明  
- **谓词表示**：  
  - P1: ∀x (Natural(x) → Integer(x))  
  - P2: ∃x (Natural(x) ∧ Even(x))  
  - 结论: ∃x (Integer(x) ∧ Even(x))  
- **子句集**：  
  {¬Natural(x) ∨ Integer(x), Natural(a), Even(a)}  
- **归结**：  
  [¬Natural(x) ∨ Integer(x)] + [Natural(a)] → Integer(a)  
  Integer(a) + Even(a) → ∃x (Integer(x) ∧ Even(x)) ✅  

#### 9. 挑瓜决策树（根节点=纹理）  
- **计算信息增益**：  
  - 纹理：清晰(8/17)、稍糊(5/17)、模糊(4/17)  
  - I(纹理) = 0.998 - [ (8/17)*I(清晰) + ... ] = **0.381**（最大）  

#### 10. 谓词逻辑归结  
- **知识库**：  
  ```  
  P1: ∀x (Pass(x) ∧ Win(x) → Happy(x))  
  P2: ∀x (Study(x) ∨ Lucky(x) → Pass(x))  
  P3: ¬Study(zhang) ∧ Lucky(zhang)  
  P4: ∀x (Lucky(x) → Win(x))  
  ```  
- **归结证明Happy(zhang)**：  
  1. ¬Happy(zhang)（假设结论假）  
  2. P3 → Lucky(zhang)  
  3. P4 + Lucky(zhang) → Win(zhang)  
  4. P2 + ¬Study(zhang) → Pass(zhang)  
  5. P1 + Pass(zhang) + Win(zhang) → Happy(zhang)（矛盾）✅  

#### 11. 贝叶斯最优分类  
- **假设概率**：  
  P(h1|d) ∝ P(d|h1)P(h1) = 0.5×0.3 = 0.15  
  P(h2|d) ∝ 0.3×0.3 = 0.09  
  P(h3|d) ∝ 0.2×0.4 = 0.08  
- **分类投票**：  
  h1预测正类，h2、h3预测负类  
  加权投票：正类权重=0.15，负类权重=0.09+0.08=0.17  
  → **输出负类（－）**  

#### 12. 老实人归结  
- **知识库**：  
  A说：B∧C说谎 → ¬B ∧ ¬C  
  B说：A∧C说谎 → ¬A ∧ ¬C  
  C说：A∨B说谎 → ¬A ∨ ¬B  
- **归结矛盾**：  
  C的真话导致A、B矛盾 → **C是老实人，A、B说谎**
